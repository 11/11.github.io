EduScape

EduScape was a awarded web application created in 24 hours at HackPSU Spring, 2015 winning a Microsoft honorable mention runner up prize. EduScape provides a competitive, gamified addition to traditional grading. With a goal to tie into products that implement LTI, it was intended that classrooms and university's may use the tool to improve a student's drive to do well. EduScape goes beyond just providing a point system for grades. It also reveals anonymous student rankings in classes, and allows students to challenge one another to see who can get the better grade with the added risk of wagering your points. Implementation was built using Microsoft Azure (running Ubuntu 14.04 LTS), PostgreSQL, Nginx, Java Play Framework 2.3.1less

Google Feud

Google Feud is an web-app game that borrows the Family Feud game show theme and web scraping technology that pulls information from Google's auto-complete data. The game play involves the user selecting a topic of their choce, where then the game generates a funny half-completed google query. With this information, the application scrapes google for the top 8 search result for the user to guess. If guessed correctly, the player gets to move on to until their streak runs out.

Base-sis

Base-sis is a music and graphics application that takes in a audio file as input and generates visuals along with the base notes to the song that is playing. Base-sis uses the Minim Music API to where it runs a fast fourier transform on the song data to seek out what frames of music store a lower frequency when compared to other frames. When the Minim API detects a base note, light meshes are rendered by utilizing LibGDX's Box2DLights API.

GG-Engine

GG-Engine is a complete Java 2D Platformer game engine written using LibGDX and the Box2D physics frameworks. GG-Engine was designed with developers and content creators in mind. The engine itself is completely open source and was structured to allow developers to adopt and build right on top of the code base at a high level. As for designers, the engine comes with support for GUI tools such as Tiled Map Editor, and Box2D to allow for non-programmers to design levels and physics related puzzles.
