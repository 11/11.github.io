These following list is a of projects that I have worked on heavily over the course of my college years hacking at hackathons and or in my free time.

  • Actively Maintained
  • Under Development

Actively Maintained

  • usacs.rutgers.edu is the general Rutgers computer science club website.
  • Jaci (Just Another C++ Inpterpreter) is designed to be a live REPL for C++

Under Development

  • GGEngine is a general purpose game engine developed for 2D platformers in mind.

Finished

  • Triangle Algorithm Visualization is research project that visualizes the Triangle Algorithm by Professor Kalantari.
  • Txtris is a twilio based hack that reads SMS messages from an audience and channels their messages as input into a single game of Tetris.

Hackathon

  • a2b is an app that can get directions through SMS.
  • mkide is simple code editor that allows you to rearrange code from a file through a GUI.
  • IRL Pong is a mobile app that allows you to play pong where your phone is the paddle.
  • Base-sis is an app that visualizes dubstep music.
  • Dynamo is a chrome extension that dynamically agenda based on how long it takes you to accomplish tasks
  • EduScape is a webapp that allows students to wager credits to compete is an educational duel.
